# 🏛️ E-Auction Platform

A full-stack online auction platform that enables users to create auctions, place bids, manage profiles, and track transaction history in real time.

## 📌 Overview

E-Auction is a web-based auction management system developed using HTML, CSS, JavaScript, Node.js, Express.js, and SQLite. The platform provides a secure and interactive environment where users can participate in online bidding and manage auctions efficiently.

---

## 🚀 Features

### User Management

* User Registration and Login
* Profile Management
* Password Change Functionality
* Account Balance Management

### Auction Management

* Create Auctions
* Upload Multiple Item Images
* Set Starting Price
* Configure Auction Duration
* Categorize Auction Items
* View Auction Details

### Bidding System

* Real-Time Bidding
* Highest Bid Tracking
* Auction Winner Determination
* Sold and Unsold Auction Tracking

### Financial Features

* Deposit Virtual Funds
* Balance Tracking
* Transaction History

### User Dashboard

* Profile Statistics
* Auctions Created
* Auctions Won
* Auctions Sold
* Auctions Unsold
* Total Bids

### UI Features

* Responsive Design
* Dark/Light Theme
* Animated User Interface
* Image Gallery Support
* Interactive Navigation

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/auction-website.git
cd auction-website
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Application

```bash
node server.js
```

### 4. Open Browser

```text
http://localhost:3000
```

## 🗄️ Database Information

This project uses SQLite as the database.

The database file (`auction.db`) is not included in the repository and will be generated automatically when the application starts.
